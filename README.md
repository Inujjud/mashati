# موقع مشاتي السعودية — نسخة بدون مجلدات (للرفع المباشر)

كل الملفات في مستوى واحد بدون مجلدات، فتقدرين ترفعينها كلها مع بعض على GitHub مباشرة.
الموقع تفاعلي ومتصل بقاعدة بيانات Supabase، وفيه لوحة تحكم للإدارة.

## الملفات (ارفعيها كلها)

```
index.html        ← الصفحة الرئيسية
about.html        ← من نحن
services.html     ← خدماتنا (المقاولات + التراث)
projects.html     ← المشاريع (تُحمَّل من قاعدة البيانات + فلتر)
process.html      ← خط سير المشروع
contact.html      ← تواصل معنا (يحفظ الطلبات)
admin.html        ← لوحة التحكم (تسجيل دخول بكلمة مرور)
style.css         ← تصميم الموقع
admin.css         ← تصميم لوحة التحكم
config.js         ← الإعدادات (Supabase، واتساب، كلمة المرور)
main.js           ← تفاعلات الواجهة + ربط Supabase
admin.js          ← منطق لوحة التحكم
Kufam.woff2       ← خط Kufam (للويب)
Kufam.otf         ← خط Kufam (الأصلي)
README.md         ← هذا الملف
logo.png          ← (تضيفينه أنتِ) شعارك بدون خلفية
```

## المزايا

- زر واتساب عائم وزر «طلب عرض سعر» في كل الصفحات (تُحفظ الطلبات في قاعدة البيانات وتظهر في لوحة التحكم).
- نموذج تواصل يحفظ الطلبات.
- المشاريع تُحمَّل ديناميكياً مع فلتر حسب التصنيف.
- تسجيل تلقائي لزيارات الصفحات لعرضها في التحليلات.
- أيقونات SVG فقط (بدون إيموجي)، وخط Kufam في العناوين.

## لوحة التحكم (admin.html)

ادخلي على `admin.html` وأدخلي كلمة المرور (الحالية: `Mm332211` — غيّريها من `config.js`).

| القسم | الوظيفة |
|------|---------|
| التحليلات | إجمالي الزيارات، عدد الطلبات، عدد المشاريع، والزيارات حسب كل صفحة |
| المشاريع | إضافة / تعديل / حذف مشروع مع رفع صورته |
| صور الموقع | رفع وتغيير صور (الواجهة، من نحن، صور الخدمات) — تتحدّث مباشرة |
| طلبات عروض الأسعار | عرض كل الطلبات مع رابط واتساب مباشر للعميل |

> ملاحظة أمنية: التحقق من كلمة المرور يتم في المتصفح فقط (حماية بسيطة، ليست كاملة)، ومفتاح Supabase العام يسمح بالكتابة للجمهور. مناسب لموقع تعريفي. للحماية الكاملة فعّلي Supabase Auth.

## الشعار (اللوقو)

ضعي شعارك **بدون خلفية** باسم `logo.png` بين الملفات، ويظهر تلقائياً في كل الصفحات. إن لم يوجد، تظهر أيقونة بديلة.

## النشر على GitHub Pages

1. ارفعي كل الملفات (مع `index.html`) في جذر المستودع.
2. من إعدادات المستودع: Settings → Pages → Source: main / root.
3. سيظهر رابط الموقع خلال دقائق.

## الإعدادات (config.js)

من هذا الملف تتحكمين في: رابط ومفتاح Supabase، رقم الواتساب ورسالته، رقم الهاتف، كلمة مرور لوحة التحكم، ومجلد التخزين.
